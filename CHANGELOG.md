# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.1] - 2023-08-20

### Fixed
- 修复已删除帖子仍被推荐的缓存问题
- 增强日志记录，便于调试
- 改进搜索结果的可见性检查

## [1.2.0] - 2023-08-15

### Added
- 添加用户使用限制功能
- 改进中文关键词提取算法
- 优化搜索结果排序

## [1.1.0] - 2023-07-30

### Added
- 增加DeepSeek AI支持
- 修复UI兼容性问题
- 改进错误处理

## [1.0.0] - 2023-06-30

### Added
- 初始版本发布 | Initial release
- AI聊天功能，支持多种AI服务提供商 | AI chat functionality with support for multiple AI service providers
- 论坛内容智能索引功能 | Forum content intelligent indexing
- 每日使用次数限制功能 | Daily usage limits
- 权限控制系统 | Permission control system
- 多语言支持（中文/英文） | Multi-language support (Chinese/English)
- 响应式UI设计，支持明/暗主题 | Responsive UI design with light/dark theme support
- 管理后台配置界面 | Admin configuration interface
